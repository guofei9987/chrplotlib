# chrplotlib
Plot graphs using chars

install:
```bash
$pip install chrplotlib
```

For the current developer version:
```bash
git clone git@github.com:guofei9987/chrplotlib.git
cd scikit-opt
pip install .
```

## example

```python
data = 10 * np.random.rand(20)
label = range(20)
```
